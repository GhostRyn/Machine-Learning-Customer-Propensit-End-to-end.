## 🚀 Cara Menjalankan

### 1. Clone Repository
```bash
git clone https://github.com/GhostRyn/ml-customer-propensity-kelompok-6.git
cd ml-customer-propensity-kelompok-6
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Jalankan Streamlit App
```bash
streamlit run app.py
```

Buka browser → `http://localhost:8501`

### 4. Jalankan Notebook di Google Colab
1. Buka link Google Colab di atas
2. Upload `training_sample.csv` dan `testing_sample.csv` ke Colab
3. Klik **Runtime → Run all**
